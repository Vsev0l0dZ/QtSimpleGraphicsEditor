# QtSimpleGraphicsEditor
Simple Graphics Editor

Простейший графический редактор, 
в котором можно рисовать прямоугольники и эллипсы с контуром:

- разного цвета
- разной толщины
- разным стилем пера

Параметры фигур сохраняются в таблице.
